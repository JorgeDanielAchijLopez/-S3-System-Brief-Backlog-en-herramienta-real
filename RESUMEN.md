# InvControl Pro

InvControl Pro es un sistema de gestión de inventarios, ventas diarias y conciliación contable diseñado para pequeños y medianos negocios.

El sistema permite:

- Registrar productos con código único.
- Registrar ventas diarias con actualización automática del inventario.
- Realizar conciliaciones entre inventario físico y sistema.
- Registrar facturas contables manualmente.
- Visualizar ganancias y pérdidas por rango de fecha.

## Objetivo

Centralizar el control operativo y financiero del negocio en una sola plataforma para mejorar la precisión del inventario y la toma de decisiones.
